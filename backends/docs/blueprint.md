# **App Name**: NextAuth Portal

## Core Features:

- User Registration: Next.js UI for user account creation, handling input validation.
- User Login/Logout: Next.js UI for user authentication (login form) and session management (logout).
- Password Encryption Placeholder: Integration point for secure password handling with a backend service. This Next.js app does not handle server-side password encryption directly.
- Role-Based UI: Client-side rendering adjustments and visibility based on user roles fetched from a backend.
- Protected Routes: Client-side routing and middleware in Next.js to restrict access to authenticated users.
- Dashboard Page: A basic dashboard page accessible to logged-in users, displaying user-specific information.
- Admin Panel UI: A dedicated section of the UI accessible only to users with the 'ADMIN' role, presenting basic administration links or components.

## Style Guidelines:

- Primary color: A deep royal blue (#1E6EDC) to convey trust and professionalism. (HSL: 220, 80%, 50%)
- Background color: A very light, desaturated blue tint (#ECF0F7) to maintain a clean and spacious interface. (HSL: 220, 25%, 95%)
- Accent color: A soft, light violet (#B499E6) used for interactive elements and highlights, providing subtle contrast. (HSL: 250, 60%, 70%)
- Headline and body font: 'Inter', a grotesque-style sans-serif for a modern, objective, and highly readable look.
- Utilize simple, outline-style icons from a library like Font Awesome or Lucide to maintain a clean and professional appearance.
- Employ a responsive, grid-based layout with generous white space to enhance clarity and user experience on various screen sizes.
- Subtle, non-disruptive transitions for page loads and hover effects on interactive elements, ensuring a smooth user flow.
